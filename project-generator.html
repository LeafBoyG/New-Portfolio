SVG Shape Generator

Interactive Generative Art Tool (Vanilla JS & SVG)

The Concept

I wanted to make a background for my browser so I decided this is how could get what I wanted. The goal was to build a browser-based "playground" where users could generate assets for web backgrounds or print, without needing complex design software.

The Engineering

Unlike many tools that use libraries like p5.js, I built this using Vanilla JavaScript and the DOM API to manipulate SVGs directly. This ensured the tool remained lightweight and responsive.

Key technical challenges I solved:

  • Dynamic SVG Rendering: Using document.createElementNS to programmatically build SVG paths (arcs, circles, eye-shapes) based on a grid matrix.
  • Image Export: Implementing a conversion pipeline that serializes the SVG XML, draws it onto an HTML5 Canvas, and exports it as a high-res PNG.
  • Interactivity: I added touch-gesture support, allowing mobile users to "swipe" to randomize colors or shapes.

The Design System

The UI is designed to be unobtrusive. I built a collapsible control panel that floats above the artwork. The application features a theming engine that allows users to define custom color palettes or choose from presets like "Neon Glow" or my favourite "Ocean Deep".

To improve the user experience, I integrated Shepherd.js to create an interactive onboarding tour for first-time users.
